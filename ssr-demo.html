SSR Demo Instructions

To test Server-Side Rendering with Routing and Styling:

  1. Install dependencies (if needed): npm install
  2. Build and run SSR server: npm run ssr
  3. Visit http://localhost:3000

Features to Test:

  • Server-Side Rendering: View page source to see pre-rendered HTML
  • Hydration: Interactive elements work immediately
  • Client-Side Routing: Navigate between pages without refresh
  • CSS-in-JS: Styled components with server-rendered styles
  • State Management: Counter and input fields maintain state

Alternative Commands:

  • Build only: npm run build:ssr
  • Run server only: npm run start:ssr
